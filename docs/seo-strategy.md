### Positive patterns to keep
- Direct category-led navigation (users can find products fast)
- Strong trust framing around supply reliability and coverage area
- Immediate contact options (phone/WhatsApp) above the fold

### Negative patterns to avoid
- Cluttered homepages with weak visual hierarchy
- Thin category pages with low search intent matching
- Missing educational content hubs that target long-tail queries

## Implemented On-Page SEO System

- Unique titles/meta descriptions by page intent
- Bilingual URLs via query-language strategy (`?lang=ar`) with alternate hreflang tags
- Canonical URLs and crawl directives
- Semantic heading architecture and internal links
- JSON-LD schemas by page type
  - Organization, LocalBusiness, WebSite + SearchAction
  - FAQPage (home)
  - ItemList (products)
  - BreadcrumbList (resource detail)
- Dynamic sitemap.xml and robots.txt
- Resource hub for long-tail query capture

## 3 SEO Edge Features Included

1. Intent-driven resource architecture from markdown content (fast indexing + easy publishing)
2. Multi-schema graph strategy (entity clarity + rich result eligibility)
3. Keyword cluster routing across commercial pages + informational guides for topical authority
